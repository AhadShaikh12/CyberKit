# CyberKit

Modern OSINT & Threat Intelligence Platform featuring VirusTotal, Shodan, Censys, AI-powered analysis, and cybersecurity utilities.

## Features

- **VirusTotal Integration**: Analyze files and URLs for malware and other threats.
- **Shodan Integration**: Search for internet-connected devices and vulnerability intelligence.
- **Censys Integration**: Gather detailed network and certificate information.
- **AI-Powered Analysis**: Leverage AI for threat intelligence insights and summary.
- **Cybersecurity Utilities**: Various tools and utilities for OSINT and scanning.

## Vercel Deployment Settings

- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`
